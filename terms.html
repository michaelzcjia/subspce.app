Description of Service

The Service enables users to create their own subspaces on their browser. A subspace refers to a collection of browser windows and the tabs within those windows. The Service allows the users to create their own sessions with a date-timestamp for the start and end of it, and attach a subspace of their choosing to it. When the session starts, the collection of browser windows and tabs in the attached subspace will automatically open for the user.

Third Party Application Integration

Subspce integrates with Google Calendar allowing the user to seamlessly create subspace sessions while scheduling events on Google Calendar. On Google Calendar, upon creating an event, the user will be given the option of adding a subspace, and if selected, the event will automatically be added to the Subspce Chrome Extension as a scheduled session. Similarly, if a user creates a session in the Subspce extension, a corresponding event will be created on their Google Calendar. To provide this functionality, we request permissions to read, and write to the users Google Calendar. These privileges are used solely for the outlined functionality of the Service.
